Het keyword if wordt altijd gevolgd door een conditie, en vervolgens een dubbele punt. :
Het keyword elif heeft een nieuwe conditie. Het code block dat hierbij hoort wordt uitgevoerd als alle voorgaande condities false zijn, en de conditie die bij de elif hoort True is.
Het keyword else heeft geen conditie. Het codeblock dat hierbij hoort wordt uitgevoerd als alle voorgaande condities niet waar zijn.
Er wordt altijd maar één code block uitgevoerd per if.
Een dubbele punt geeft aan dat er een code block volgt.
