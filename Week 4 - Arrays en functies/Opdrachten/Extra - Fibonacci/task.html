Omdat de fibonacci-reeks gebaseerd is op de voorgaande resultaten in de reeks, is het een goede kandidaat om recursie op toe te passen. Recursie is een onderwerp dat veel dieper is dan de scope van deze lessen, maar voor degenen die extra uitdaging zoeken is het een mooi onderwerp.

Recursie is het aanroepen van een functie binnen de functie zelf. Hiervoor zijn er altijd twee cases in de functie:
De base case:
De case waar de functie een statische waarde teruggeeft
De recursive case:
De case waar de functie een call naar zichzelf met andere argumenten teruggeeft. De recursive case moet op een manier altijd uiteindelijk terugkomen bij de base case.

Functies

Functies stellen, net als operators, acties voor. We zijn functies al tegengekomen bij het toevoegen en verwijderen van data in een array.

We kunnen een functie aanroepen door de naam van de functie te typen, gevolgd door ronde haken:

functie()

Tussen deze ronde haken kunnen we de argumenten van de functie zetten. De argumenten van een functie zijn de stukken data die we als input meegeven aan die functie. Argumenten worden gescheiden van elkaar door kommas.

functie(arg1, arg2)

Soms heeft een functie een uitkomst. Niet alle functies hebben een uitkomst. Wanneer een functie een uitkomst heeft, kunnen we deze opslaan in een variabele:

var = functie(arg1, arg2)

We zullen het nu over een aantal handige functies hebben, en hoe deze in gebruik genomen kunnen worden.

Input

De input functie wordt gebruikt om de gebruiker om input te vragen. Wanneer we deze functie aanroepen, stopt ons programma, en wacht het tot de gebruiker iets in typt en op enter drukt.
Deze functie heeft dan als uitkomst hetgene wat de user heeft ingetypt.
De uitkomst van input is altijd een string.

Deze functie heeft een (optioneel) argument. Het type van dit argument is een string. Deze string wordt aan de gebruiker weergegeven wanneer deze om input wordt gevraagd. Gebruik dit argument om aan de gebruiker uitleg te geven over wat er ge-input moet worden.

Casting

Casting is het veranderen van data van een bepaald type naar een ander type. Casting is niet altijd mogelijk. Het kan alleen gebeuren wanneer Python een waarde van het type waar we naartoe casten in de variabele die we casten kan vinden.

Om te casten gebruiken we een andere functie per data type:
Cast naar Functie
integer int(arg)
string str(arg)
float float(arg)
boolean bool(arg)

Wanneer casting mogelijk is, hebben deze functies als uitkomst de waarde van arg van het type waar we naartoe hebben gecast.

Print

We zijn de print functie al meerdere malen tegengekomen in de lessen. We gebruiken deze functie wanneer we iets in de console weer willen geven.

De print functie neemt 1 argument, hetgene wat we willen printen. Dit argument mag ieder type zijn.

De print functie heeft geen uitkomst. Het enige wat het doet is het weergeven van een visuele vorm van wat we in de functie stoppen in de console

Len

De len functie geeft ons de lengte van een verzameling.

len neemt 1 argument, de verzameling waarvan we de lengte willen weten. Dit argument moet dus een verzameling zijn. LET OP, een string is niets anders dan een verzameling van karakters

len heeft als uitkomst de lengte van de verzameling die we als argument meegaven. De uitkomst van len is dus altijd een integer.

Tot nu toe hebben we het gehad over bestaande functies. Deze functies zitten al in python. We kunnen natuurlijk ook onze eigen functies maken. Hiervoor gebruiken we het keyword def

def

Het keyword def geeft aan dat er een functie definitie volgt. Def wordt altijd gevolgd door de naam van de functie, en dan ronde haken. Tussen de ronde haken zetten we dan de argumenten die de functie inneemt.

Vervolgens is er een dubbele punt. Dit geeft weer aan dat er een code block komt. In dit code block beschrijven we dan wat er gebeurt als de functie aangeroepen wordt.

return

We hebben nu gezien hoe we onze eigen functies maken, maar nog niet hoe we deze een uitkomst geven. Hiervoor gebruiken we het keyword return

Wat er volgt achter het keyword return is de uitkomst die wordt teruggegeven wanneer de functie aangeroepen wordt.
