Functies

Functies stellen, net als operators, acties voor. We zijn functies al tegengekomen bij het toevoegen en verwijderen van data in een array.

We kunnen een functie aanroepen door de naam van de functie te typen, gevolgd door ronde haken:

functie()

Tussen deze ronde haken kunnen we de argumenten van de functie zetten. De argumenten van een functie zijn de stukken data die we als input meegeven aan die functie. Argumenten worden gescheiden van elkaar door kommas.

functie(arg1, arg2)

Soms heeft een functie een uitkomst. Niet alle functies hebben een uitkomst. Wanneer een functie een uitkomst heeft, kunnen we deze opslaan in een variabele:

var = functie(arg1, arg2)

We zullen het nu over een aantal handige functies hebben, en hoe deze in gebruik genomen kunnen worden.

Input

De input functie wordt gebruikt om de gebruiker om input te vragen. Wanneer we deze functie aanroepen, stopt ons programma, en wacht het tot de gebruiker iets in typt en op enter drukt.
Deze functie heeft dan als uitkomst hetgene wat de user heeft ingetypt.
De uitkomst van input is altijd een string.

Deze functie heeft een (optioneel) argument. Het type van dit argument is een string. Deze string wordt aan de gebruiker weergegeven wanneer deze om input wordt gevraagd. Gebruik dit argument om aan de gebruiker uitleg te geven over wat er ge-input moet worden.
