Arrays

In Python heten arrays lists. In de opdrachten worden zowel array als list goed gerekend.
Een array is een data type. Het is een verzameling van andere data. De array zelf is alleen een container waar andere data in geplaatst kan worden.
Om een array aan te maken, gebruiken we blokhaken: [ ]
Als we dan data of variabelen tussen deze blokhaken zetten, wordt de array aangemaakt met deze data er al in.

Data aanroepen

Om data aan te roepen uit een array, gebruiken we weer de blokhaken. We noemen eerst de array waar we de data uit willen oproepen, direct gevolgd door de blokhaken. Tussen de blokhaken zetten we dan de index van het element dat we op willen vragen.

Index begint met tellen bij 0.
1e element 2e element ne element laatste element
index 0 1 n - 1 (lengte van array) - 1

Wanneer we data aanroepen uit een array, kunnen we hier alles mee doen wat we ook met een variabele kunnen.

Data overschrijven

Net als met variabelen, kunnen we de operator = gebruiken om een nieuwe waarde te plaatsen op de gegeven positie in de array:

Array[index] = NieuweWaarde

Data toevoegen

Wanneer we data willen toevoegen aan een array, hebben we in Python hiervoor twee mogelijkheden. Deze mogelijkheden zijn functies van array. Wat een functie is leer je later deze week. Omdat de functies bij de array zelf horen, moeten we een punt . gebruiken om de functie toe te passen op de array.

Data achteraan toevoegen

Om data achteraan de array toe te voegen, gebruiken we de functie append. Deze functie neemt 1 argument, namelijk de data om toe te voegen. Argumenten worden tussen ronde haakjes achter de naam van de functie geplaatst:

MijnArray.append(NieuweData)

De bovenstaande code voegt de waarde van de variabele NieuweData achteraan in de array MijnArray toe.

Data op een andere positie toevoegen

Om data op een bepaalde positie in de array toe te voegen, gebruiken we de functie insert. Deze functie neemt 2 argumenten.

Het eerste argument is de index waar we willen dat de nieuwe data komt te staan.
Het tweede argument is de data die we willen toevoegen.

Argumenten worden tussen ronde haakjes, gescheiden door kommas achter de naam van de functie geplaatst:

MijnArray.insert(NieuweIndex,NieuweData)

De bovenstaande code voegt de waarde van de variabele NieuweData op de positie NieuweIndex in de array MijnArray toe.
